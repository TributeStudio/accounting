- **Goal:** Allow uploading/attaching Master Service Agreements to individual clients/projects.
- **Functionality:**
  - Extract and store key contract details automatically (using Gemini?):
    - Hourly Rates
    - Invoice Scheduling / Billing Cycles
    - Payment Terms
  - Use this data to auto-populate invoices and set default rates for projects.
  - "Knowledge Base" for each client where these documents live.
